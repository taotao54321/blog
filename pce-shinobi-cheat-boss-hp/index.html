忍 (PCE) ボスのHPを減らす裏技

2023-07-30

ボス戦中に I, II, Select, Up, Left を押しつつ Run を押すとボスのHPが 1 減る。 そのまま Run を連打すればすぐにボスを倒せる。

該当コード: バンク 3 論理アドレス $AA3A
